# BeYou Care - Landing Page

## ARCHIVOS INCLUIDOS

- beyoucare-landing.html  (Página principal)
- BeYouCare.jpeg          (Logo oficial)
- README.md               (Este archivo)

## INSTRUCCIONES DE USO

### Para visualizar la página:
1. Descarga todos los archivos a la misma carpeta
2. Abre beyoucare-landing.html con tu navegador
3. La página se mostrará con el logo y todo el contenido

### Para publicar online:
1. Sube ambos archivos (HTML + imagen) a tu servidor
2. Mantén los archivos en la misma carpeta
3. Accede a través de tu dominio

## CARACTERÍSTICAS

- Diseño responsive (se adapta a móvil, tablet y desktop)
- Logo oficial de BeYou Care integrado
- 5 ponentes con sus credenciales completas
- Horario detallado del evento
- Animaciones suaves al hacer scroll
- Secciones: Hero, Información, Programa, Ponentes, CTA, Footer
